# Zhuchenko-Volodymyr-UID_lab1

Десктопний застосунок **«Калькулятор кредита та депозита»**, розроблений для лабораторної роботи №1  
з курсу **«Розробка інтерфейсів користувача»**.

---

## Опис проєкту

Застосунок реалізує інтерфейс для розрахунку **кредиту** та **депозиту**.

Програма надає користувачу можливість:
- вводити фінансові дані через графічний інтерфейс
- отримувати результат розрахунку у текстовому полі
- зберігати результати у **локальний файл**
